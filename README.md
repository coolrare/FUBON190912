# Angular 進階實作班起始專案

```shell
git clone https://github.com/coolrare/angular-realworld-advanced-starter.git
cd angular-realworld-advnced-starter
npm install
npm start
```

# 練習用的後端

```shell
git clone https://github.com/coolrare/realworld-backend.git
cd realworld-backend
npm install
npm start
```
